Title: JRuby
Presented By: Mark Volkmann, OCI
About the Presentation:

JRuby is an implementation of the Ruby language on the Java Virtual Machine. There are three primary motivations for using JRuby.

  • Use Java Libraries (for example, Swing) from code written in Ruby syntax.
  • Use Ruby libraries (for example, ActiveRecord) from code written in Java syntax using either the Bean Scripting Framework (BSF) or the JSR 223 Scripting API.
  • Get a faster implementation of Ruby (not yet faster, but likely will be soon).

This presentation will provide an overview of JRuby. It will also walk through examples of using Java libraries from JRuby and using Ruby libraries from Java.
