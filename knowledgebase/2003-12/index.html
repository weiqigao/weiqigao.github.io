Title: Aspect-Oriented Programming (AOP) - Even More Depth!
Presented By: Bob Lee, SBC
About the Presentation: Bob will show how we can apply what we've learned from EJB and use Java AOP frameworks to break loose of the limitations of OO programming alone. He will concentrate on real world use cases rather than the logging, metrics, and debugging examples we've all surely grown tired of.
About the Presenter Bob Lee is a coauthor of Manning's book "Bitter EJB". He is also a technical architect for SBC.
