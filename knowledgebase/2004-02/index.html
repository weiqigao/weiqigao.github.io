Title: J2EE 1.4 Web Services
Presented By: Mark Volkmann, OCI
About the Presentation:

The J2EE 1.4 specification defines specific ways to implement and invoke web services in Java. This talk will show examples of these, mainly focusing on invoking web services. We'll cover the SOAP with Attachments API for Java (SAAJ) and Java API for XML Remote Procedure Calls (JAX-RPC) APIs.

About the Presenter:

Mark Volkmann has been a software consultant for Object Computing, Inc. in St. Louis since 1996. He has been developing software in Java since 1995. For the past few years he has focused on using XML with Java. As a consultant, Mark has assisted many companies with Java and XML application development. Mark also has created and teaches many courses in Java and XML. They include: Introduction to Java, Java Syntax for Non-C Programmers, Java Programming, Advanced Java Programming, Creating Graphical User Interfaces Using Java (Swing), eXtensible Markup Language (XML), XML Programming Using Java and Web Services Using Java. Mark is a member of the St. Louis Java User Group steering committee and a regular presenter for that group. He has written for XML Journal, presented at XML DevCon conferences and presented at a No Fluff Just Stuff Java symposium..
