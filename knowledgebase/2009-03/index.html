Title: Android
Presented By: Eric Burke, OCI
About the Presentation:

Android is Google's open source mobile platform for phones and (soon) GPS devices, netbooks, and all sorts of other gadgets. Using familiar Java syntax, programmers can create mobile applications and sell them on the Android Marketplace. In this month's talk, Eric will demonstrate how to write Android applications using the Android SDK and Eclipse plugins. We'll see how to create GUIs, respond to touch gestures, interact with maps, animate graphics, and more.
About the Presenters:

Eric Burke is a Principal Software Engineer with Object Computing, Inc. He enjoys blogging, creating comics, programming and home improvement.
