Title: Introduction to SonarQube
Presented By: Chris Vogel
About the Presentation:

SonarQube ("Sonar") is an open-source platform for continuous inspection of code quality to perform automatic reviews with static analysis of code to detect bugs, code smells, and security vulnerabilities in many coding languages. In this presentation, we will look at how to integrate Sonar into the build process, including incorporating it into a Jenkins build, and use it to improve your application code.
About the Presenter:

Chris Vogel is a Technical Architect at Edward Jones, where he has worked for over twenty-two years. He has worked with Java since 1997. The majority of his career has been spent helping set the application development direction and supporting teams by defining application architecture, supporting developer tools (including Sonar) and mentoring.
