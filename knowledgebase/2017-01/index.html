Title: Spring Boot
Presented By: Jenny McLaughlin
About the Presentation:

The Spring Framework is one of the most popular Java frameworks. Spring Boot is the next chapter of the Spring Framework. In this presentation, we will demonstrate how to create enterprise-ready applications by using spring boot and all its enterprise -ready features , from zero config files, faster development, to easy deployment.
About the Presenter:

Jenny McLaughlin has many years of Java and Spring experience across various industries. She is currently a Platform Architect at Pivotal.
