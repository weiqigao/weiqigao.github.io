Title: Grails
Presented By: Jeff Brown, OCI
About the Presentation:

Grails aims to bring the "coding by convention" paradigm to Groovy. Grails is an open-source web application framework that leverages the power of a dynamically typed language to greatly simplify web development. Developers with experience using action based or component based web frameworks for Java will be amazed at the agility and ease of use that Grails brings to the table. This session will provide a brief introduction to the Groovy language and a detailed review of Grails.
About the Presenter:

Jeff Brown is a Principal Software Engineer with Object Computing Inc. based in St. Louis Missouri and has been with OCI since 1997. In addition to consulting work Jeff has developed and maintains numerous Java related courses taught through OCI and Washington University's CAIT.

Jeff is a member of the St. Louis Java User Group steering committee and a regular presenter for that group.
