Java Data Objects (JDO)

By Jeff Brown, OCI

About the Presentation

Java Data Objects (JDO) is an API for transparent data persistence in Java. JDO provides an object oriented approach to persistence while hiding most of the persistence details from the application developer. The presentation will discuss the JDO API, compare JDO and JDBC, discuss several JDO implementations and will work through several working examples.

About Jeff Brown

Jeff Brown is a Senior Software Engineer with Object Computing Inc. In addition to Java consulting work, Jeff teaches and maintains a number of OCI's Java training courses including JDBC For Portable Database Access.
