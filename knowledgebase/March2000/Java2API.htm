    Examples

     
    Example
    Description
    PrintPreview.java
    		 This class represent the print previewer used in all the exercices.
    It is based from chapter 22 of the book "Swing" by M. Robinson and P. Vorobiev. I modified it
    to handle Pageable jobs.
    SimplePrint.java
    		 This example prints a simple string on a page using the Graphics context.
     SimplePrint2D.java
    		 This example prints a simple string on a page using a Graphics2D context.
     PrintableJPanel.java
    		 This example takes a JPanel and implement the Printable interface to print the JPanel.
     SimpleFrame.java
    		 This example use the PrintableJPanel and show how to print Swing component.
     SimplePrint2DBook.java
    		 This example show how to use the Pageable interface and the Book class to print a document.
     SimpleClipping.java
    		 This example show how to use the clipping area of the Graphics2D context.
