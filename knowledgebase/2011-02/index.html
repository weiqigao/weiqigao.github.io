Title: RabbitMQ
Presented By: James Carr (CARFAX)
About the Presentation:

RabbitMQ is a robust messaging broker utilizing the open AMQP protocol that is both high performance and easy to use. In addition to digging into the details of how RabbitMQ works there will also be an in-depth demonstration of using the new spring-amqp module as well as tying different languages and platforms together using AMQP based messaging.
About the Presenters:

James Carr is a passionate developer who is a contributor to a few open source projects with a keen interest in message oriented middleware.
