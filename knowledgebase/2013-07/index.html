Title: Get Up and Running With GitHub
Presented By: Brent Beer GitHub
About the Presentation:

Maybe you've heard of GitHub and haven't started playing with it yet, or maybe you've created a profile but haven't added much to it. What's stopping you from showing the world what you know and love?

In this talk, we'll walk through common entry points to getting started in open source communities as well as advantages to putting our own projects on GitHub.
About the Presenter:

Brent Beer has been a passionate user of Git and GitHub for a number of years beginning in university and later moving out to San Francisco to work as a web developer. He now enjoys spending his time helping people learn to use Git and GitHub to their full potential as a member of the GitHub Training team.
