Title: JUnit: The Other Stuff
Presented By: Charles SharpOCI
About the Presentation:

JUnit 4 has been out since 2006 but aside from @Before, @Test and Asserts how much of JUnit is really used?

This presentation discusses and presents code samples of some less commonly used features of JUnit 4.11 and throws in a quick tour of the Hamcrest matchers for good measure.
About the Presenter:

Charles Sharp is a Principal Software Engineer at Object Computing, Inc., a leading software engineering firm providing consulting for software architecture, design, and implementation. He has used JUnit for Java testing in various domains since 1999 and teaches the Java Testing with JUnit class at CAIT.
