J2EE Design Patterns

By Sharath Sahadevan

About the Presentation

As J2EE applications mature, a number of design patterns have been identified as best pratices in the industry. This presentation will discuss different design patterns like a Front Controller, Value Objects, Session Facade, Data Access Object, etc. Knowledge of these design patterns will facilitate development of better J2EE applications.

About Sharath Sahadevan

Sharath Sahadevan works on a team that develops and maintains the eservice web site at MasterCard International. eservice is a secure site available to members. This site integrates with a number of back-end systems and presents reports to users. It is developed using J2EE.

Sharath has been working with Enterprise Java technology for about 3 years. He is a Sun certified Enterprise Architect. When not working, he enjoys spending time with family and playing cricket, chess, basketball and tennis.
