Title: Apache Hadoop
Presented By: Tom Wheeler
About the Presentation:

As systems become increasingly automated, we're generating more information -- and doing it faster than ever before. Apache Hadoop makes it possible to store and analyze "Big Data" in order to gain valuable insight into customer behavior, important trends, and new opportunities.
Apache Hadoop, originally inspired by Googleâ€™s GFS and MapReduce papers, is an open source framework offering scalable, distributed, fault-tolerant data storage and processing on standard hardware.
During this talk, Tom will provide a solid overview of Apache Hadoop, explaining topics like HDFS replication, data locality, and MapReduce processing, from the ground up. He'll also cover a number of complementary tools that help you integrate Hadoop with existing systems and simplify typical analytical tasks.
About the Presenter:

Tom Wheeler's career spans more than fifteen years in the communications, biotech, financial, healthcare, aerospace and defense industries. Before joining Cloudera in 2011, he was a software engineer, technical trainer, and occasional system administrator.
