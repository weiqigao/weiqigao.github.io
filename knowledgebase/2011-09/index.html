Title: Spring 3.0 and 3.1: New Java-Based Configuration Options
Presented By: Chris Hardin (OCI)
About the Presentation:

Spring 3.0 introduced a new way of configuring beans that affords many of the same advantages as the traditional XML configuration but uses pure Java code instead. This Java-based configuration style provides great flexibility and ease of use while keeping Spring configuration separate from the core application code. Spring 3.1 is on the horizon and will introduce even more support for this style of configuration as well as new concepts such as bean profiles and an environment abstraction. We will walk through quickly setting up an XML-free configuration for a small application and explore some of the new features coming in Spring 3.1.
About the Presenters:

Chris Hardin is a Senior Software Engineer with Object Computing, Inc. (OCI). Chris has worked for or contracted with companies around St. Louis, including Boeing, Express Scripts, and Sisters of Mercy. She also develops and teaches courses on the Spring framework for OCI and the Center for the Application of Information Technology (CAIT) at Washington University.
