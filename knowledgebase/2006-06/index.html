Title: JavaServer Faces (JSF)
Presented By: Jay Meyer, Harpoon Technologies
About the Presentation:

JavaServer Faces (JSF) simplifies development of sophisticated web application user interfaces, primarily by defining a user interface component model tied to a well-defined request processing life cycle. This presentation will cover the basics of JSF as well as the new technologies that are leveraging the power of JSF, MyFaces, Seam, Shale, Facelets, and others.
About the Presenter:

Jay Meyer is a partner and software engineer at Harpoon Technologies, a St. Louis software consulting company. Jay has written software for 14 years for companies in St. Louis such as Union Pacific, A-B, Boeing and several smaller software companies. Jay is a proponent of open source systems and recently completed JBoss Advanced Developer certification. Jay has a Masters degree in Computer Science from Washington University in St. Louis. He can be reached at jmeyer@harpoontech.com.
