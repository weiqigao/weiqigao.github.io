Title: RESTful Web Services using JAX-RS and Jersey
Presented By: Brian Gilstrap
About the Presentation:

This talk with describe JAX-RS (an API) and Jersey (a reference implementation), which provide support for building RESTful web services in Java. After a brief discussion of REST, we will look at how to use JAX-RS for a variety of tasks, and see how it can save time, lines of code, and complexity when building web services that can deliver content in formats tailored to client's needs.
About the Presenters:

Brian Gilstrap is a Principal Software Engineer with Object Computing, Inc. With over twenty years spent building many kinds of software with many different technologies, he is always looking for faster and better ways to build effective, reliable, and secure software.
