Title: Lambda Expressions in Java 8
Presented By: Weiqi Gao
About the Presentation:

Lambda expressions (or closures) is a feature that is present in many programming languages. Talk about adding lambda expressions to the Java programming language started many years ago, and three competing proposed implementations (BGGA, etc.) were floating around at one time. The latest Java road map would include lambda expressions in Java SE 8, to be released in 2013. The language specification changes is happening in JSR 335 while the implementation changes for OpenJDK is happening in Project Lambda.

In this talk, we will cover the lambda expression related changed to Java from a Java programmers perspective. We will look at work that is done in Project Lambda, and go through both the language changes (which has basically been done for some time) and library changes (which is still on going).

If you are curious about any of the following topics:

  • Functional interfaces
  • Lambda expressions and their types
  • Default methods in interfaces
  • Effectively final variables
  • Method and constructor references
  • Additional functional methods to the Collections interfaces
  • The Stream interface
come to this talk.
