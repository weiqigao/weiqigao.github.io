Title: Migrating Beyond Java 8
Presented By: Dalia Abo Sheasha
About the Presentation:

Are you thinking about migrating your application to Java 11 or newer and wondering what is going to break? Come to this session to learn about the common pitfalls when moving from Java 8 and how to resolve them. The goal from this session is to prepare you for the issues you will encounter during your own application migration and introduce tools that will make the process easier.
About the Presenter:

Dalia Abo Sheasha is the development lead for the IBM WebSphere Migration Tools. Her team develops several migration tools that deal with a variety of scenarios including Java SE migrations and on-premise to cloud migrations. Dalia has worked on WebSphere for 6 years starting her career working on Enterprise JavaBeans and Java Persistence API where she contributed to the open-source projects: EclipseLink and OpenJPA.
