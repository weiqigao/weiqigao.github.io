    • getOutputStream, getWriter, setContentType

  • Several useful HTTP constants are defined in this interface

    • see JavaDocs

  • Important methods

    • sendError(int statusCode)
      • see following page
    • sendRedirect
      • send an alternate URL to the client
    • addCookie
    • several other methods
      • see JavaDocs
