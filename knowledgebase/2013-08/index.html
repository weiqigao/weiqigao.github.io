Title: Real-time Web Services with REST Hooks
Presented By: James Carr Zapier
About the Presentation:

Today offering a solid API is quickly becoming an important prior ity for businesses and other services to allow interconnectivity with other popular services. Exposing an API is easy, but designing a GREAT API that is easy t o use can be extremely difficult. In this talk I'll go over some common patterns for good API desig n and delve into the concept behind REST Hooks, a mechanism for cre ating subscribable webhooks that allow for service consumers to stop po lling for new data.
