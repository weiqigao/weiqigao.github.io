Title: How Functional Programming Made Me a Better OO Developer
Presented By: Jessica Kerr (Stealth Mode Startup)
About the Presentation:

Do you want to be a better Java developer? Find out what functional programming has to teach us about OO--without learning a functional language. Learn some lessons and principles of the functional style that apply directly to OO programming tasks. This talk will not teach functional programming; instead, it skips straight to the parts that apply to Java. We'll find principles that we're already starting to use in Java, patterns both old and new, and different ways of thinking about problems. Take home some new techniques that can make Java code clean, efficient, and more maintainable.
About the Presenter:

Jessica Kerr has written Java for Daugherty, Monsanto, ESI, Amdocs, and several other St. Louis companies during her twelve years as a developer. Lately she's big into the user group community, speaking locally and at conferences like CodeMash and KCDC.
