Title: JMX
Presented By: Adam Quan
About the Presentation: As more and more mission-critical enterprise applications are being built and deployed on J2EE infrastructures, better administration, monitoring and management of J2EE applications, as well as underlying network and system infrastructures are needed. Java Management Extension (JMX) defines a management architecture, APIs, and management services that provide a comprehensive management framework for the Java environment. This talk will provide a technology overview of the JMX technology. Adam will be focusing on JMX concepts, architecture and real-world JMX application usages.
About the Presenter Adam Quan is a Technical Architect with Edward Jones. Adam started working with Java in 1997, and has been focusing on J2EE technologies for the past several years. Adam earned an M.S. degree in Computer Science from Washington University in St. Louis.
