Title: Tuscany - Apache SOA Framework
Presented By: Jeffrey Guo, Accelrys
About the Presentation:

Tuscany is a project to bridge the gap between language-specific application component implementation technologies and higher-level SOA (service-oriented architecture) concepts and design approaches. It simplifies the composition of SOA-based systems through a set of frameworks and solutions defined via the Service Component Architecture (SCA) specification. Service Data Objects (SDO) further simplify the data tier programming of SCA by unifying data programming across data source types and providing support for common application patterns.

This talk will give you an overview of the Service Component Architecture (SCA) and its related data service technologies SDO and DAS (Data Access Service). Also discussed in this talk are some useful extensions available in the SCA Java implementation (implementation type containers and transport/protocol extension bindings).
About the Presenter:

Jeffrey Guo is a Senior Solutions Consultant for Accelrys, Inc., a life science software and service solutions provider. He is passionate about delivering enterprise solutions to critical research informatics problems in the biotechnology and pharmaceutical industry with Java and open source technologies. Jeffrey has extensive development experience in J2EE and SOA.
