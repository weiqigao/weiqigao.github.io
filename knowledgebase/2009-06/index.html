Title: TDD With Groovy
Presented By: Matt Taylor, OCI
About the Presentation:

This presentation will give a brief introduction to the Groovy language before jumping directly into a live build of a Groovy application using Test-Driven Development. It will be code intensive, and give some examples of advanced usage of the Groovy language, including metaprogramming, closures, and DSL's. The whole point of the presentation application will be to evaluate this syntax:

  • 20.meters.to.miles
  • 100.fahrenheit.to.kelvin
  • 60.kilometers.to.inches

The app. will understand it and return the proper result. We'll just have a simple lookup table for the conversion, but the fun part will be the metaprogramming involved. The source code is available at http://github.com/rhyolight/groovy-unitconverter/tree/master.
