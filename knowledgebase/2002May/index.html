About Tom Reilly

Tom Reilly is a Principal Engineer at Macromedia where he is lead architect of the web tier of the JRun. Tom is also responsible for the design and implementation of JRun's JSP engine and serves on the servlet, JSP and XML Java Community Process expert groups. In previous lifetimes Tom worked on writing OpenGL programs in Java and the Mosaic web browser at NCSA.
