Title: Project Lombok: How I learned to love NOT coding
Presented By: Denny Slover
About the Presentation:

A brief introduction to Project Lombok and how it will keep your code clean, consistent and saving you time! Project Lombok is a boilerplate code generator that will make your code look cleaner, consistent and save you time. Unlike the IDE code generators, this will create the code at compile time with some simple annotations. Once you use this framework, you will never want to write boilerplate code again.
About the Presenter:

Denny Slover is a Tech Lead at MasterCard and currently working on the MasterPass wallet. He has been working in the St Louis IT industry got about 10 years. He has been using Spring and Java related frameworks for most of that time. He has attended JUG meeting for years and learned a lot and now he wants to share his knowledge with others.
