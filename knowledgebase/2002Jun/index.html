June 13, 2002

BEA WebLogic Server 7.0 Technical Overview

By Alexander Berry, BEA

About the Presentation

Technical presentation on the latest release of WebLogic Server, version 7.0. This focus of this presentation is WebLogic Server's industry leading capabilities in the following areas:

o Built-in Scalability and Reliability
o Built-in Integration Infrastructure (based on J2CA)
o Pluggable Security Framework (based on JAAS)
o Built-in High Availability JMS Messaging
o Built-in Web Services (XML,SOAP)
o Wireless and Mobile Commerce
o J3EE Standards Compliance and Leadership (EJB 2.0,MDB, etc.)
o Distributed Management.

Presentation Material: power point format
