Title: Java Concurrency By Example
Presented By: Eric Burke, OCI
About the Presentation:

If you bought a computer in 2006, it probably has a dual-core CPU. In 2007 quad core CPUs will be common. Intel even predicts systems with dozens or hundreds of cores in the next decade. Is your code ready for this revolution?

In this talk, we'll see many of the Java 5 concurrency features in action. Through code reviews and live demonstration, we'll learn about features like:

  • atomic variable classes
  • Executors, Callables, thread pools, and thread factories
  • Lock objects
  • concurrent collections
  • Swing thread safety
  • barriers, latches, semaphores
  • interrupting tasks gracefully
About the Presenter:

Eric Burke is a Principal Software Engineer with Object Computing, Inc. in St. Louis.
