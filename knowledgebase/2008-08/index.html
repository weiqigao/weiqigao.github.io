Title: Java Classloaders
Presented By: Charles Sharp, OCI
About the Presentation:

Most of us first encounter Classloaders while trying to figure out ClassNotFoundExceptions.

Most of us first curse Classloaders while trying to figure out ClassCastExceptions.

This presentation gives the basic rules for Classloading(and why you get those pesky exceptions), how a Classloader contributes to the Java Virtual Machine's wonderfully flexible architecture, and a short introduction to writing your own Classloaders.

About the Presenters:

Charles Sharp has learned to deal with( but never really forgiven) the Classloader not finding a ClassDef when the class is in plain sight. He is a Senior Software Engineer at Object Computing, Inc., a Sun Authorized Java Center and member of the Object Management Group in St. Louis.
