January 9, 2003

Five different speakers presented on six different Java-related software tools. All of them are open source.

Jeff Brown from OCI discussed JarSpy, a utility for examining and reporting on the contents of JAR files.
Presentation Material: JarSpy.pdf

Presentations

Brian Coyner from OCI discussed xdoclet, a utility for generating files from Java source files based on special tags in javadoc comments. This is especially useful for generating EJB-related files.
Presentation Material: XDoclet.pdf

Mark Volkmann from OCI discussed Jalopy and CheckStyle. Jalopy is a tool for reformatting Java source files to comply with specified coding styles. CheckStyle is a tool for reporting on Java source files that do not meet specified guidelines, such as the number of lines of code in a method.
Presentation Material: JalopyCheckstyle.pdf

Bob Lee from OCI discussed jAdvise (http://crazybob.org), a framework for inserting method interceptors into plain Java objects at runtime. jAdvise can be used to implement everything from logging to implicit transaction management to transparent persistence. The jAdvise source code forms the foundation of the upcoming JBoss AOP framework.
Presentation Material: JAdvise.pdf


Alan Wang from Connectria discussed CruiseControl, a tool for automating continuous builds, unit test suite execution and reporting of results.
Presentation Material: CruiseControl.pdf
