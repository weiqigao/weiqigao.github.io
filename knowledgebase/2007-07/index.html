Title: JavaFX Script
Presented By: Weiqi Gao, OCI
About the Presentation:

At JavaOne 2007, Sun Microsystems announced the JavaFX product family. The JavaFX product family will leverage the ubiquity of the Java platform to deliver high-impact content to a variety of platforms including desktop, mobile devices and set-top boxes.

JavaFX Script is one of two initial members (the other being JavaFX Mobile) of this product family. It is a a highly productive scripting language for content developers to create rich media and content.

According to https://openjfx.dev.java.net/JavaFX_FAQ.htm: "JavaFX Script is a declarative, statically typed programming language. It has first-class functions, declarative syntax, list-comprehensions, and incremental dependency-based evaluation. It can make direct calls to Java APIs that are on the platform. Since JavaFX Script is statically typed, it has the same code structuring, reuse, and encapsulation features (such as packages, classes, inheritance, and separate compilation and deployment units) that make it possible to create and maintain very large programs using Java technology."

In this presentation, we will explore JavaFX Script from a scripting language point of view, covering its basic language constructs, highlighting some of it not-so-familiar features such as type-inference, list-comprehension, incremental and lazy evaluations, and the duration operator (for animations). We will also cover the JavaFX wrappers for Swing and Java2D classes that are available in the JavaFX Script runtime library.
About the Presenters:
