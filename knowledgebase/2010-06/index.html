Title: Google Collections
Presented By: James Carr (OCI)
About the Presentation:

Google Collections provides a wide variety of features that make working with collections more efficient and fun. James will show examples of using collection literals, handy utility operations and some of the functional programming constructs that the library provides.
About the Presenters:

James is a contractor in the St.Louis area that shares a passion for software craftsmanship and has enjoyed software development as a hobby since he wrote his first program in Basic on the Tandy Color Computer 3 way back in 1988.
