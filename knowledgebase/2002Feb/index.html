February 14, 2002

IBM WebSphere Application Server 4.0

By Virgil Work, Jr., Software IT Specialist, IBM Corp.

About the Presentation

This presentation focuses on WebSphere Application Server 4.0. The presentation explores IBM WebSphere Application Server, an e-business application deployment environment built on open standards-based technology. The presenter explains the product's features, standards compliance, editions, application deployment and analysis tools.

About Virgil Work, Jr.

I have worked for over 20 years with IBM. In my career I have worked on a variety of hardware and software from mainframes to PC's. My current position is Software IT Specialist for WebSphere and Application Development Tools.
