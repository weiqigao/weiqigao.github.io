Title: Android Memory Management
Presented By: Lee McCauley
About the Presentation:

Memory management for mobile platforms is key to performance and user adoption. For apps that can be running for days at a time, there's nothing worse than having your user's device grind to a slow, painful death because of your application. Lee will describe techniques for tracking an application's memory usage and fixing identified memory leaks in an Android app. Among other things, he will touch on how Android shares memory between processes and the implications of anonymous inner classes, asynchronous callbacks, and strong vs. weak references.

About the Presenter:

Lee McCauley is a Senior Software Engineer and Agile Practitioner at World Wide Technologies, Asynchrony Labs. He has worked with Java since 1991 and Android since 2013.
