J2SE 1.5 Overview

Un-Typesafe Collections

  • Before J2SE 1.5, collections are not typesafe

    List driverNamesList = new ArrayList();

driverNamesList.add("Castroneves");
driverNamesList.add("DeFerran");
driverNamesList.add("Hornish Jr.");

Iterator it = driverNamesList.iterator();
while (it.hasNext()) {
    String driverName = (String)it.next();
    System.out.println(driverName);
}

  • The (String) cast inside of the while() loop is necessary because any type of object may be returned from the call to it.next()

  • List, ArrayList and all other collection types may hold any type of Object

  • Adding anything other than a String to driverNamesList is allowed, but a ClassCastException will be thrown from the while() loop
