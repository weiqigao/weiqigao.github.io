J2SE 1.5 Overview

Typesafe Enums (Cont'd)

  • Enums have a public field called VALUES of type java.util.List

  • VALUES contains each value in the enum

    for(IndyCarChassis chassis: IndyCarChassis.VALUES) {
    System.out.println(chassis);
}

  • Each value in the enum has a public final int field called "ordinal"

  • Each value in the enum has a unique ordinal value

    • assigned based on the order of declaration

    • starts with zero
