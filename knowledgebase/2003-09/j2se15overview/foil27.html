J2SE 1.5 Overview

Typesafe Collections

  • J2SE 1.5 Typesafe Collections allows for a more specific expression of collection types

    List <String> driverNamesList = new ArrayList <String> ();

driverNamesList.add("Castroneves");
driverNamesList.add("DeFerran");
driverNamesList.add("Hornish Jr.");

// no need to cast here...
String driverName = driverNamesList.get(1);
System.out.println(driverName);

  • Now our List is constrained to only allow String objects

  • Adding anything other than a String to driverNamesList generates a compile time failure
