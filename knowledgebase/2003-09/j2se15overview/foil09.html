J2SE 1.5 Overview

Variable Argument Lists

  • Variable argument lists provide a syntax to support defining a method that accepts a variable number of arguments

  • Example:

    public void doSomething(String arg1, String... varArgs) {
}

  • This method accepts a String argument, followed by any number of additional String arguments (including zero)

  • The type of varArgs is a String array

    • any Object type may be specified

    • any primitive type may be specified
