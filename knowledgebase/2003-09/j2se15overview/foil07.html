J2SE 1.5 Overview

Autoboxing Primitives

  • The java.lang package contains "type wrapper" classes for each of the 8 primitive types defined by the language spec

    • Integer

    • Boolean

    • Float

    • etc...

  • A common use of these wrapper types is to store primitive values in a collection

    List myList = new ArrayList();

myList.add(new Integer(42));
myList.add(new Integer(2112));
