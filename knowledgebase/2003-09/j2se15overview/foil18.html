Typesafe Enums (Cont'd)

  • A class or interface containing valid values could be created

    public interface IndyCarChassis {
    public static final String DALLARA = "Dallara";
    public static final String G_FORCE = "G-Force";
}

  • Now these constants could be used as arguments to the IndyCar constructor

    IndyCar andretti = new IndyCar(IndyCarChassis.DALLARA);

  • The IndyCar constructor still needs to deal with invalid values because there is no way for the compiler to enforce that the caller used the constants in IndyCarChassis
