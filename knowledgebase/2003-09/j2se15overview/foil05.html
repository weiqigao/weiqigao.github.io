Static Imports

  • Static members in other classes are usually prefixed with a class name

    double valueOfPI = Math.PI;

  • Static imports remove the need for the class prefix

    import static java.lang.Math.*;

public class Test {
    public static void main(String[] args) {
        
        // PI is a static field in the Math class
        System.out.println("PI = " + PI);

        // max(int, int) is a static method in the Math class
        int max = max(args.length, 3);
        System.out.println("max = " + max);
    }
}
