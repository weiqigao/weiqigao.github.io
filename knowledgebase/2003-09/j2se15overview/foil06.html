J2SE 1.5 Overview

Static Imports (Cont'd)

  • Often interfaces are written for the purpose of definining a group of constants

  • A class may implement the constant interface and then refer to the constant names without the class or interface name prefix

  • Static imports are a cleaner mechanism for solving this problem

  • The class that is using the constants doesn't need to implement the interface containing the constant definitions

  • The class inheritance isn't complicated just to gain easy access to constant values
