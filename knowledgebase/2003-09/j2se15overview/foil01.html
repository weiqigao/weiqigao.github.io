J2SE 1.5 Overview

What Is J2SE 1.5

  • J2SE 1.5 is the next major release of the Java 2 Standard Edition (J2SE)

  • J2SE 1.5 introduces a number of new language features

  • Aside from assertions introduced in J2SE 1.4, the Java language has not changed much

  • First public beta of J2SE 1.5 is expected to be available late 2003

  • Many of the new features that may be included in J2SE 1.5 are geared toward simplifying code
