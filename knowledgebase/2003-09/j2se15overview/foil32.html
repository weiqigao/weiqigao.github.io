J2SE 1.5 Overview

Bounded Parameters

  • Parameterized types may be "bounded"

    public class MyParameterizedType <T extends Person> {
    //...
}

  • In this case, the parameterized type must extend or implement Person

  • Person may be a class or an interface

    // this is ok if Employee extends or implements Person
MyParameterizedType<Employee> mpt =
    new MyParameterizedType<Employee>();

// this is NOT ok
MyParameterizedType<Vector> mpt =
    new MyParameterizedType<Vector>();
