J2SE 1.5 Overview

Parameterized Types

  • Generics may be used to create "Parameterized Types"

    public class MyParameterizedType <T> {

    private T t;

    public void setValue(T t) {
        this.t = t;
    }

    public T getValue() {
        return t;
    }
}

  • Note that "T" is not a defined type, "T" is the parameter

  • "T" may be used anywhere a type would normally be found

    • field and variable types

    • method arguments

    • return types
