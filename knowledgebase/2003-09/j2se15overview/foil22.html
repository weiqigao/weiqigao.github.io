Typesafe Enums (Cont'd)

  • Arguments may be passed to the enum constructor

  • Enums may contain methods and fields like any other "class"

    public enum IndyCarChassis {
   DALLARA("Dallara"), G_FORCE("G-Force");

   private String chassisName;

   private IndyCarChassis(String chassisName) {
       this.chassisName = chassisName;
   }

   public String getChassisName() {
       return chassisName;
   }
}
