J2SE 1.5 Overview

Parameterized Types (Cont'd)

  • Whan an instance of MyParameterizedType is created, the type that "T" represents must be specified

    MyParameterizedType<String> myStringWrapper =
    new MyParameterizedType<String>();

  • In this case, "T" is specified to be of type String

  • Once declared this way, compile time checks enforce that the type is not violated

  • For example:

    // this is fine...
myStringWrapper.setValue("Penske Racing");

// but this is not...
myStringWrapper.setValue(new Integer(6));

// this is fine...
String value = myStringWrapper.getValue();

// but this is not...
Vector v = myStringWrapper.getValue();
