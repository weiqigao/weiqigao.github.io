J2SE 1.5 Overview

Typesafe Enums (Cont'd)

  • J2SE 1.5 enums provide a solution that calls for less code and is more consistent with enums in other programming languages

    public class IndyCar {

    public static enum Chassis { DALLARA, G_FORCE };

    private Chassis chassis;

    public IndyCar(Chassis chassis) {
        this.chassis = chassis;
    }
}

  • Now a member of IndyCar.Chassis may be passed as an argument to the IndyCar constructor

    IndyCar castroneves = 
    new IndyCar(IndyCar.Chassis.DALLARA);
