J2SE 1.5 Overview

Bounded Parameters (Cont'd)

  • Multiple constraints may be specified

    public class MyParameterizedType <T extends Person & Expense> {
    //...
}

  • Any number of interfaces may be specified this way

  • Only one class may be specified

  • If a class is specified, it must be the first type in the list

  • An ampersand is used to delimit types

    • comma is used to separate parameters

  • A type passed as an argument must meet all of the type specifications

    // Employee extends Person and implements Expense...
MyParameterizedType<Employee> mpt = 
    new MyParameterizedType<Employee>();
