J2SE 1.5 Overview

Typesafe Enums (Cont'd)

  • In pre J2SE 1.5 Java, a typesafe enumeration could be simulated

    public class IndyCarChassis {

    public static final IndyCarChassis DALLARA = 
        new IndyCarChassis("Dallara");

    public static final IndyCarChassis G_FORCE = 
        new IndyCarChassis("G-Force");

    private final String myName;

    private IndyCarChassis(String name) {
        myName = name;
    }
}

  • IndyCar constructor could accept a IndyCarChassis object as an argument instead of a String object

  • The compiler can prevent invalid values from being passed as arguments to the IndyCar constructor
