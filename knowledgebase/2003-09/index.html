Title: New Features In J2SE 1.5
Presented By: Jeff Brown, OCI
About the Presentation: J2SE 1.5 will introduce several new features. This presentation will cover a wide range of these features including generics, auto boxing, enhanced for loop construct, metadata and static imports. Of the new features coming in J2SE 1.5, generics are attracting the most attention. The first beta release of J2SE 1.5 is scheduled for release in late 2003.
About the Presenter Jeff Brown is a Principal Software Engineer with Object Computing Inc. and has been with OCI since 1997. Jeff started doing Java development in 1996 after several years working with C, C++ and other languages. In addition to Java consulting work Jeff also authors and teaches Java courses for OCI including Java Programming, Memory Management In Java, Java Syntax for Non-C Programmers, JDBC For Portable Database Access, and other courses. Jeff has been a member of the St. Louis Java user Group steering committee for several years and has offered numerous presentations to the group.
