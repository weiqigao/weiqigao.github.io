Title: Java Flight Recorder and Java Mission Control
Presented By: Dave DiFranco
About the Presentation:

Did you know that the Oracle JVM comes with free enterprise-grade profiling tools?

Java Flight Recorder (JFR) is a lightweight tool for collecting performance data from a running JVM. Java Mission Control (JMC) is a user interface for analyzing performance data, including data collected by JFR.

This session will cover how to use JFR to collect performance data off-line and on-line. It will cover how to analyze the results using JMC. This will include how to find bottlenecks in memory allocation, method profiling, thread usage, socket usage, and file I/O.

No prior knowledge of performance profiling is necessary.
About the Presenter:

Dave DiFranco is a software engineer at Oracle, working on SOA Suite and the Integration Cloud Service. Dave has been a java developer for over 15 years including work at BEA, MetaMatrix, and Tripos. Dave uses JFR and JMC regularly to solve production issues.
