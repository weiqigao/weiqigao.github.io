Title: Spring Data JPA
Presented By: Denny Slover
About the Presentation:

This talk will cover the basics of Spring Data JPA, including configuration, dynamic finders, auditing strategies, as well as reasons why you want to start using Spring Data for your project.
About the Presenter:

Denny Slover is a Tech Lead at MasterCard and currently working on the MasterPass wallet. He has been working in the St Louis IT industry for about 10 years. He has been using Spring and Java related frameworks for most of that time. He has attended JUG meeting for years and learned a lot and now he wants to share his knowledge with others.
