Java Messaging Service

By Cindy Pilling, Anderson Technologies

About the Presentation

The presentation provides an overview of the Java Messaging Service. Specifically, how to use JMS's different messaging models as well as the features of a MOM. In addition, considerations when debating how to incorporate JMS into the enterprise.

About Cindy Pilling

A Java Consultant with Anderson Technologies, Inc. Cindy's Java experience includes technical leadership and architecting of both Client and Server side enterprise level Java applications. When not promoting Java and associated technologies, Cindy enjoys cycling, flying, golf and spending time with family and friends.
