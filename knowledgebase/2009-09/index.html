Title: Wicket
Presented By: Luther Baker
About the Presentation:

"Much like Object Relational Mapping (ORM) technologies such as Hibernate and TopLink try to address the impedance mismatch between relational databases and object-oriented (OO) Java programming, Wicket aims to solve the impedance mismatch between the stateless HTTP protocol and OO Java programming." [Taken from Wicket in Action, Dashorst and Hillenius]

This presentation will introduce Wicket and its programming model in the context of JEE web frameworks. We will motivate the case for Wicket and explore it's underlying principles and goals as well as provide examples and comparisons of the Wicket approach as it relates to other popular models of web development.

About the Presenters:

Luther is a Senior Application Architect at Suddenlink Communications in St. Louis, MO.
