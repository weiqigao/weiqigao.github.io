Title: Amazing Android
Presented By: Eric Burke (Square)
About the Presentation:

This month, Eric Burke shares tips and tricks learned while creating Square for Android. We'll see how to create efficient, beautiful UIs with custom textures, animations, and styles. The talk is geared towards all skill levels, presented through a series of small code samples and the resulting user interface.
About the Presenters:

Eric Burke is a local programmer who has been creating user interfaces for 15+ years. He wrote three O'Reilly books and now works as an Android Client Engineer with Square, Inc.
