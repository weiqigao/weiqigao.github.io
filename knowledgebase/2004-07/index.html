Title: Apache Velocity
Presented By: Naeem Bari, Cequel3
About the Presentation:

"Velocity is a Java-based template engine. It permits anyone to use the simple yet powerful template language to reference objects defined in Java code" (from http://jakarta.apache.org/velocity/index.html).

This presentation will introduce developers to Velocity, and how to use it for web applications. We will write a small servlet for demonstration purposes. Naeem will also discuss how his framework uses Velocity. Oh, and we will be using the Eclipse IDE, so developers will be able to see it in action.

About the Presenter:

Naeem Bari is currently the Director of Technology at Cequel3, a cable & wireless management firm. Naeem has been involved in developing java based frameworks since 1999, when he wrote his first framework, MerlinWeb. Since then he has revised and rewritten his framework multiple times. The current generation is built around Velocity, Apache's outstanding templating engine. Naeem has a Masters in CS from Washington University and hails from Pakistan.
