October 10, 2002

Eclipse IDE

by Gareth Davies, Daugherty Systems

About the Presentation

The Eclipse project is an open-source, OS-independent platform for integrating a variety of development tools. It supports many languages and products, but its core competency is Java. It thus constitutes a viable alternative to other free Java IDEs and may even challenge some commercial IDEs. Gareth will summarize the architecture of the Eclipse platform and then describe and demonstrate some of the key productivity-enhancing features of this rapidly emerging IDE.

About Gareth Davies

Gareth Davies has an MS in Nuclear Engineering and has been a software consultant with Daugherty Systems in St. Louis since 1996. For most of the last decade he has worked as an application architect, primarily with Microsoft technologies - especially Visual Basic, ASP and COM (MTS) - using successive versions of the Visual Studio IDE. Over the last couple of years he has become progressively more involved with Java development. For his current client, he is using Eclipse to develop a web application with (amongst other things) XML, Oracle, Tomcat and Jakarta Struts.
