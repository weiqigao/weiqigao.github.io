Title: Spring External Configuration Server
Presented By: Denny Slover
About the Presentation:

With this Spring Cloud Config, the application configuration can be separated and centralized to repository to make the management easier. The configuration is then loaded at startup for each environment. This also allows changes to the configuration as the application is running and a hot refresh of the Spring configured beans without any interruption to the service. This talk will introduce the project, the code needed to use it and how to structure the configuration file to best leverage this framework.
