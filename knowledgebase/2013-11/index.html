Title: Deploying Application With Heroku
Presented By: Clint Shryock Heroku Inc
About the Presentation:

Heroku is a platform explicitly designed for development and deployment of applications, running your apps in the clouds with ease and a lot of purple.

Run anything using existing Java frameworks or JVM languages like Scala and Clojure. Deploy with Tomcat, Jetty, or Netty. Build with popular frameworks like Spring, Grails, Play!, and Tapestry.

Forget Servers and Focus on what matters - building your application - not on system administration or server provisioning. Heroku is a service that you can begin using for production apps, right now.
