Title: JUnit 4
Presented By: Charles Sharp, OCI
About the Presentation:

JUnit 4 was the first major release of JUnit in five years. It replaces the naming conventions used in JUnit 3 with annotations that specify the tester's intents.

This presentation discusses the features of JUnit 4.4. It shows what the JUnit 3.8 features look like in 4.4 and shows how to quickly migrate a JUnit 3.8 test to a JUnit 4.4 test. It also discusses the new features found in 4.4 and how they can be used in unit testing your Java code.
About the Presenters:

Charles Sharp is a Senior Software Engineer at Object Computing, Inc., a Sun Authorized Java Center and member of the Object Management Group in St. Louis. He has used JUnit for Java testing in various domains since 1999. He enjoys making assertions, some of which pass.
