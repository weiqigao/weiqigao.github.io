Title: AJaX
Presented By: Mark Volkmann, OCI
About the Presentation:

AJaX stands for Asynchronous JavaScript and XML. It is used to create more dynamic web applications where data displayed on a web page is obtained by using JavaScript to make asynchronous calls to server-side code that typically return the data as XML. The calls are made using a JavaScript class called XMLHttpRequest.

This talk will walk through an AJaX application that uses a JavaScript library called Sarrisa. The application retrieves data from a MySQL database in a fashion similar to Google Suggest.
