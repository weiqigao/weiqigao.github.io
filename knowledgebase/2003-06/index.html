Title: Struts
Presented By: Greg Elliott, OCI
About the Presentation: The Struts framework is rapidly becoming the industry standard for web applications using Java Servlets and Java Server Pages. Initially conceived by Craig McClanahan, Struts is currently sponsored by the Apache Foundation under the Jakarta Project for open source Java solutions. The framework reduces the time it takes to build web applications from scratch by providing the invisible underpinnings all professional web applications need to survive. The framework is based on the Model 2 architecture, a variation of the proven Model-View-Controller design paradigm. Enterprises can benefit from their developers’ enhanced ability to focus on implementing business requirements rather than on implementing the underlying framework first.
About the Presenter Greg Elliott has been a software consultant for 6 years. He currently works for Object Computing, Inc. in St. Louis, and is one of the lead developers on an enterprise-level supplier management tool for Boeing. Prior to becoming a software developer, Greg held various positions in corporate finance. Greg has a Bachelors from Southern Illinois University in German and Russian, and earned a Masters in Business Administration from the American Graduate School of International Management in Glendale, AZ.
