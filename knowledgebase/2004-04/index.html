Title: Enterprise Component Design
Presented By: Kent Miller, Daugherty Systems
About the Presentation:

This presentation will start with a high level overview of how to create the necessary environment to build reusable enterprise components. This discussion will includes the necessary skills, Leadership, Processes and Methodologies, Requirements, Function Point Identification, TDD checkpoints, Accurate Estimating, Identification of the optimal component environment(s), Architectural Design, Construction, Automated Unit Testing, Refactoring, Regressing Testing, System Testing, User Acceptance Testing, and Implementation. Then we will delve into more detail on areas that the developers are directly responsible for. Ending with a summary of how to build re-usable enterprise components.

About the Presenter:

Kent Miller is an experienced J2EE Architect currently working at A-B. He has architectural experience in full project life cycle development. Including requirements gathering, identification of needed resources, accurate estimating, outsourcing (the true issues and costs associated with outsourcing), software development, integration with existing systems, construction, testing, and implementation.
