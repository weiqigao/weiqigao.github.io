Title: Build a Better API with GORM & GraphQL
Presented By: Zachary Klein
About the Presentation:

GraphQL is an exceptional technology that allows APIs to be more efficient and expressive. GORM is the object relational mapper used by Grails that enables developers to interact with both relational and document databases through the declaration of simple Groovy classes. This talk will discuss in depth a new GORM extension and Grails plugin that combines these two great technologies and increases developer productivity.

Outline:

  • GORM Introduction
  • GraphQL Introduction
  • Comparison of GraphQL vs REST
  • GORM-GraphQL Features
  • GORM-GraphQL Customizations
  • GORM-GraphQL Testing
About the Presenter:

OCI Senior Software Engineer, Zachary Klein, has been practicing web development since 2010 and front-end development since 2015. He's a contributor to both the Grails and Micronaut frameworks, as well as an instructor in OCI's training practice. Zachary lives in St Louis, Missouri, along with his wife, Beth, and three children.
