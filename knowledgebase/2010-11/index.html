Title: Spring Security 3.0
Presented By: Jason Ferguson
About the Presentation:

This presentation will cover the basics of authentication with Spring Security 3.0. Spring Security is a powerful and highly customizable authentication and access-control framework. It is meant to to be the de-facto standard for securing Spring-based applications.
About the Presenters:

Jason Ferguson is Java programmer on active duty with the United States Air Force. He has been a Java developer for nearly 6 years and has experience with Struts (1 and 2), Spring, Hibernate, and various other libraries.
