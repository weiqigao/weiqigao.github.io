Title: Java 3D
Presented By: Chris Gundlach, Paragenix
About the Presentation:

The Java 3D API has existed since 1999, but has not seen a great deal of interest until recently. We will start with a short overview of the history of Java 3D and other 3D technologies, touching on OpenGL and DirectX. The bulk of the talk will be technical discussion of the Java3D API, and how to utilize it in your applications. The discussion will focus on the Java 3D scene graph and how an application composes and manipulates the scene graph. Information will be presented on transformations, behaviors, performance, and how to load 3D models from popular modeling applications. The presentation will conclude with a few demos showing what is possible with Java 3D.

About the Presenter:

Chris Gundlach is President of Paragenix, a St. Louis based software development and consulting company. For the past eight years, Chris has developed applications for large organizations, primarily web based, using Java. Lately, he has become interested in software technologies for improving the user experience.
