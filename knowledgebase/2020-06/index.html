Title: Developing low latency Java applications with Hazelcast
Presented By: Sharath Sahadevan
About the Presentation:

Discuss key considerations in developing low latency applications in Java. Discuss common use cases and patterns. Discuss deployment topologies. Demo applications with real time streaming of Flight telemetry data. Review the source code and architectural choices being made as part of the design of the application. Discuss how java users can leverage Hazelcast in their applications
About the Presenter:

Sharath Sahadevan is a Solutions Architect with Hazelcast. He has previously worked for Pivotal (now acquired by VMware) and MasterCard. He has presented at the Collab Summit in 2016 on Cloud Design Patterns. He lives in St. Louis and enjoys spending time with family and playing tennis.
