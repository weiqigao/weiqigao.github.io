Title: Table-driven Acceptance Testing
Presented By: Mario Aquino, OCI
About the Presentation:

The Framework for Integrated Test (Fit - http://fit.c2.com) is a toolkit for testing application business rules using an easy to understand table-driven approach. A major benefit of this style of testing is that it provides a platform for collaboration between application developers and stakeholders to define tests that interact directly with the running application. This presentation will introduce Fit and the related FitLibrary (http://fitlibrary.sourceforge.net) project, which adds new styles of Fit tests as well as utilities for running tests as part of a continuous integration process.
About the Presenter:
