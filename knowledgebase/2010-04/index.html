Title: Code Analysis Tools
Presented By: Darryl Parls (Monsanto)
About the Presentation:

This talk will present a variety of tools that analyze Java source code. It will begin with an introduction and statistics related to manual code analysis (code reviews). Next, it will introduce automated code analysis and show basic setup and examples of:

  • FindBugs
  • CheckStyle
  • PMD
  • List of and overview of a few other tools.
  • Introduce code analysis dashboards:
  • XRadar
  • Sonar
  • Introduce (community edition of IntelliJ Idea's Inpector)
About the Presenters:

Darryl Parks is currently a Technical Lead at Monsanto, Inc. over a field sales intranet site supporting sales analysis and compensation. Darryl first started at Monsanto in 1995 as a contractor and became a full-time employee four years ago.

Darryl graduated with High Honors with a Bachelors in Managing Information Systems from Idaho State University in 1987. He has had a varied background in languages starting in RPG then Cobol, C and later into VB, MS Access, ASP and now Java. Business areas range from Issues Tracking sessions for Tiger Teams at the Idaho National Engineering laboratory and Cost Accounting.
