Title: Building JavaServer Faces Applications
Presented By: Tim Saunders, IBM Rational Software
About the Presentation:

JavaServer Faces (JSF) is a server-side technology for developing web applications with rich user interfaces. JSF technology makes it easier to build Web applications by letting users more easily assemble reusable UI components in a page, connect these components to an application data source, and wire client-generated events to server-side event handlers. The presentation will introduce JavaServer Faces and demonstrate how to build a JSF application using IBM Rational Application Developer.
About the Presenter:

Tim Saunders is an IT Specialist with IBM Rational Software. Prior to joining IBM last year, he worked at McDonnell Douglas and MasterCard as a developer and software development manager.
