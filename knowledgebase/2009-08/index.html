Title: Cloud Computing and what it means for Java/J2EE application development
Presented By: Sharath Sahadevan
About the Presentation:

This presentation is an exploration of cloud computing and what it means for Java/J2EE application development. We will explore cloud services that are offered by Amazon, Google, Yahoo and Microsoft. We will demonstrate a sample application that leverages Google's cloud infrastructure.
About the Presenters:

Sharath is a technical consultant at MasterCard International.
