Title: Javelin Stamp
Presented By: Andrew Prunicki, OCI
About the Presentation:

The Javelin Stamp is a microcontroller running a small JVM capable of executing a subset of Java. The Javelin Stamp can be used in robotics, home automation and machine control, amongst others. This presentation will introduce the Javelin Stamp:

  • Describe the basics of how to program it
  • Demonstrate a small Javelin Stamp-powered robot
  • Describe some of the Javelin Stamp's features and limitations
  • Show some sample code
  • Debug on-board the Javelin
About the Presenters:

Andrew Prunicki is a software developer with Advantage Consulting, Inc. in St. Louis. He is interested in exploring ways to make computer technology more interesting and tangible.
