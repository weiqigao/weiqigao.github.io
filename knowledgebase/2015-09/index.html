Title: Creating a REST web service server with JAX-RS and JAXB
Presented By: Jackson Thompson
About the Presentation:

REST has because a de-facto standard for creating CRUD web services. It is light-weight compared to SOAP and uses the HTTP protocol to perform it's operations. In this howto presentation we will walk through creating a set of REST services to support a cross browser javascript application. This presentation will only use JEE standard libraries, JAX-RS and JAXB, on server side handle all the web services and XML/JSON work. Added bonus, we will use the increasingly poplular AngularJS to bind our web service to browser data display components.
About the Presenter:

Jackson Thompson is a seasoned Java developer. He currently works in the Hospitality industry developing Central Reservation Systems. In the past he worked for Netscape and Sun Microsystems developing their Web Server products including the servlet container. He also supported and trained developers using the APIs for those products, including Servlets, JSP, JSF and Server Side Javascript.
