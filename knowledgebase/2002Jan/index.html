January 10, 2002

Java Web Start

By Brad Shuler, Object Computing, Inc.

About the Presentation

Using Java Web Start, developers can deploy full- featured Java 2 applications over a network. As easy to use as accessing a web page, Java Web Start provides one click activation and guarantees that users are always running the latest, centrally controlled version of the application all made possible using the new Java Network Launch Protocol (JNLP). The possibility for peaceful coexistence of multiple applications all with unique runtime environments, security needs and system properties is reason enough to be excited about this technology. This month's Java user group meeting will include a technical overview of how to set up a web site to enable deployment of a Java 2 application using Java Web Start. 

About Brad Shuler

Brad Shuler has been a software engineer at OCI since the year 2000. He holds a B.S. in Industrial Engineering from Purdue University. Brad has focused his career on user interfaces for various medical, military, and business applications. In 1992 Brad developed a powerful pre and post-processing application for computational fluid dynamics and finite element analysis for a mechanical engineering software firm in Indianapolis. Since that time Brad has focused his career on object oriented approaches and business applications. After several years as a C and C++ developer he switched to Java and now has over three years of Java experience. For the past year he has focused on using CORBA with Java. As a consultant, Brad has assisted many companies with Java and distributed application development.
