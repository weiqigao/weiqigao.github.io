Thursday, July 10, 2003
Title: JBoss
Presented By: Weiqi Gao, OCI
About the Presentation: JBoss is a popular open source Java application server that supports the J2EE 1.3 specifications. Built on top of a light-weight JMX core, JBoss provides standard J2EE services from JNDI to Servlet/JSP to EJB and JCA. Weiqi will describe how to install JBoss, configure Oracle, MySQL, and PostgreSQL data sources for use with session beans and entity beans, configure security, create a livable development environment using open source tools such as Ant, JUnit and XDoclet. He will then develop, test and deploy a simple J2EE application showcasing stateless session, entity, and message-driven beans, a web client and a rich client. Along the way he will touch upon JBoss specific topics such as the JBoss class loader architecture, JBoss specific deployment descriptors, system monotoring, and other fun things.
About the Presenter Weiqi Gao is a principal software engineer and instructor with Object Computing, Inc. Weiqi started learning Java on June 11, 1996, when he bought his first Java book (google groups: weiqi java book), and has worked on Java projects since the JDK 1.1.4 days. Weiqi is a member of the St. Louis Java User Group steering committee.
