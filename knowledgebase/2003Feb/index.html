How to Test Just About Anything

By Brian M. Coyner, Object Computing, Inc.

About the Presentation

Getting started with testing can be a frightening task. This presentation delves into ways of getting a testable development environment setup, along with showing different ways of ensuring that your code is (and stays) testable. This will show how to use mock objects to test Swing code, and scratch the surface with the do's and don'ts of server side testing.

About Brian M. Coyner 


Brian M. Coyner is an O'Reilly author and Senior Software Engineer with Object Computing, Inc. in St. Louis, MO. He has a B.S. in Computer Science from Southeast Missouri State University, and specializes in Java training and consulting. When he is not working, which is rare, Brian enjoys playing the guitar and spending time with his family.
