Title: Docker
Presented By: Steven Borrelli
About the Presentation:

Docker (http://docker.io/) makes it easy to build, share and deploy applications. Docker has seen rapid adoption since its initial release in March 2013, gaining over 10,000 stars on Github and changing the way many companies build and deploy applications.
About the Presenter:

Steven Borrelli has been working with distributed systems, clusters and cloud technology for over a decade. He founded Aster.is (http://aster.is) in January 2014 to bring next-generation technology to organizations.
