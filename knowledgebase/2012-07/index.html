Title: Rethinking Enterprise
Presented By: Ted Neward (Neudesic)
About the Presentation:

Special Summer Edition: Ted Neward on "Rethinking Enterprise"

Delivered as a keynote at CodeMash earlier this year, this talk will help every professional software developer think more clearly about what we're doing and why.

The era of the big, heavy, transactionally-oriented client/server topology, as best described by the J2EE Specification and Blueprints document, seems to be over. The era of the lightweight, transactionally-oriented client/server topology seems to be at its zenith. Is this really where we want to be when building enterprise systems? And how did we end up here, anyway? What's the new "enterprise" developer supposed to do?
About the Presenter:

Ted Neward is an Architectural Consultant with Neudesic, LLC, a software consulting firm based in the US. He is a recognized expert in both the Java and .NET communities, is interested in virtual machines, programming languages, Java/.NET interoperability and different database systems. He writes columns for Oracle TechNet, MSDN Magazine and CoDe Magazine, has published a dozen or so books, and speaks regularly in the US and Europe. Due to his rather scary resemblance to a fictional character from a Coen Brothers movie, he is sometimes known as "The Dude Of Software", which he's OK with. He resides in the Pacific Northwest with his wife, two sons, dog, four cats, eight laptops, three gaming consoles, six mobile phones, five tablets, and a scary electric bill.
