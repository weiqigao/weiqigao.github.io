JavaServer Pages Standard Tag Library (JSTL)

by Dean Wette, Object Computing, Inc

About the Presentation

JSTL encapsulates, as simple tags, core functionality common to many JSP applications. For example, instead of iterating over collections using scriptlets or proprietary iteration tags, JSTL defines a standard tag that works the same everywhere.

JSTL provides support for core iteration and control-flow features, text inclusion, I18n and formatting tags, XML- and XSLT-manipulation tags, and tags for SQL database access. It also supports an expression language (EL) - inspired by ECMAScipt and Xpath - to simplify page development without requiring expertise in the Java programming language.

JSTL is a new specification from the Java Community Process (JCP) and comes out of the JSR-052 Expert Group. The reference implementation is provided by the Jakarta Project (jakarta.apache.com), and is currently available as JSTL 1.0.2.


About Dean Wette

Dean Wette joined Object Computing, Inc. (OCI) in 1999 and has held the position of Senior Software Engineer since 2000. Since joining OCI, he has consulted to the financial and aerospace industries where he works on object oriented technology and architecture, with an emphasis on server-side Java and XML-based solutions. Previously, he was an applications analyst in the public sector in the areas of community/business development and public utilities.

Mr. Wette is also an educator for OCI. He designed and developed the Java 2 Security class, and maintains ongoing development of the Advanced Java Programming and Advanced Servlet/JSP Programming courses. In addition to these classes, he teaches Java Syntax For Non-C Programmers, Java Programming, and XML Programming Using Java. Mr. Wette also presents frequently on Java and XML topics to the St. Louis Java and XML SIGs, the University of Missouri-Rolla Computer Science Colloquia Series, and at other venues.

In 2002, Mr. Wette helped found the St. Louis XML SIG, and currently sits on the steering committee as the program director.

Mr. Wette holds a B.A. cum laude in Music from Webster University in St. Louis, has done graduate studies in Musicology and Medieval/Renaissance Studies at Washington University, and is nearing completion of a M.S. in Computer Science from the University of Missouri-Rolla. He is also an Adjunct Professor of Computer Science at the University of Missouri-Rolla Engineering Education Center (UMR-EEC) in St. Louis.
