Title: XDoclet 2
Presented By: Ross Sponholtz, Connectria
About the Presentation:

XDoclet is an open source code generation technology that allows the developer to describe attributes to java programs in JavaDoc tags. XDoclet parses source files and generates many files such as XML descriptors, source code, and web pages. Although XDoclet originated as a tool for creating EJBs, it has evolved into a general-purpose code generation engine, and is extensible for custom applications. Ross will describe XDoclet and its usage, and will give some real-world examples of applying XDoclet technology.

About the Presenter:

Ross Sponholtz is a senior System Architect with Connectria, Corp. Ross has been working with Java for several years, and has been focused on J2EE development with WebLogic. He has a BS in Computer Science from the University of Colorado, and is currently working on an MBA at the University of Phoenix.
